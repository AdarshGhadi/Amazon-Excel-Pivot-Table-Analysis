# Amazon-Excel-Pivot-Table-Analysis
Pivot table analysis of Amazon Sales data.

## Dataset
- File: Amazon_Sales_Dataset.xlsx
- Rows: 500

## Objective
Use Excel Pivot Tables to analyze sales, profit, and performance across categories, regions, and segments.

## Steps to Complete Task
1. Open dataset in Excel / Google Sheets.
2. Convert data range into a Table (Ctrl + T).
3. Create Pivot Table:
   - Total Sales by Category (Descending Order).
4. Create Pivot Table:
   - Sales by Region and Segment.
5. Add calculated column:
   - Profit Margin = Profit / Sales.
6. Use slicers for Category, Region, Segment.
7. Create Pivot Charts for visual comparison.
8. Write 5 insights below pivots.
9. Export Pivot sheet as PDF.
